# 1. Justificació

El present document descriu un model estructurat de seguiment acadèmic aplicat a l’acció tutorial en cicles formatius de Formació Professional.

Aquest model respon a la necessitat de:

- Sistematitzar la tutoria acadèmica
- Generar evidència documental objectiva
- Desenvolupar autonomia i autoregulació en l’alumnat
- Connectar formació i projecció professional
- Garantir traçabilitat i verificabilitat del procés

El projecte supera el model tradicional de tutoria informal i es fonamenta en una estructura progressiva, documentada i avaluable.

---

# 2. Finalitat del model

El projecte té com a finalitat:

- Fomentar l’autoavaluació acadèmica estructurada
- Desenvolupar capacitat d’anàlisi personal
- Millorar la planificació i gestió del temps
- Relacionar estudis amb sortides professionals reals
- Definir objectius acadèmics concrets i mesurables

Es tracta d’un model:

- Individual
- Progressiu
- Basat en evidència escrita
- Tècnicament traçable
- Avaluable amb criteris objectius

---

# 3. Estructura del projecte

El model es desenvolupa al llarg de cinc sessions distribuïdes durant el curs.

## Sessió 1 · Diagnòstic acadèmic inicial

Autoavaluació estructurada de la situació acadèmica.

Inclou:
- Coneixements previs
- Adaptació al cicle
- Anàlisi per mòduls
- Accions inicials de millora

---

## Sessió 2 · El cicle i les sortides professionals

Relació entre formació actual i possibles trajectòries professionals.

Inclou:
- Descripció del cicle
- Dues sortides professionals concretes
- Relació entre competències i ocupació
- Aspecte acadèmic a millorar

---

## Sessió 3 · Hàbits de treball i organització acadèmica

Anàlisi de l’organització personal.

Inclou:
- Planificació setmanal
- Gestió del temps
- Constància
- Compromís concret de millora

---

## Sessió 4 · Anàlisi DAFO acadèmic

Aplicació de metodologia d’anàlisi estratègica al context acadèmic.

Inclou:
- Fortaleses
- Debilitats
- Oportunitats
- Amenaces
- Síntesi amb acció concreta

---

## Sessió 5 · Síntesi i projecció acadèmica

Definició d’objectius finals.

Inclou:
- Objectius específics
- Accions calendaritzades
- Indicadors de seguiment
- Coherència amb la projecció professional

---

# 4. Metodologia

El projecte es basa en:

- Reflexió guiada estructurada
- Producció escrita obligatòria
- Evidència documental verificable
- Seguiment progressiu
- Control de versions mitjançant repositoris Git

Cada alumne disposa d’un repositori individual on es documenten totes les sessions.

Aquest sistema permet:

- Traçabilitat completa
- Control de temporalitat (commits)
- Verificació de participació real
- Generació automatitzada d’informes

---

# 5. Sistema de verificació i evidència

S’ha desenvolupat un sistema automatitzat que:

- Recorre tots els repositoris
- Verifica estructura i presència de fitxers
- Genera informes globals en format Markdown
- Converteix els informes a PDF per a Direcció

Els informes inclouen:

- Identificació d’alumne (anonimitzada o nominal)
- Contingut literal
- Incidències tècniques
- Completesa de lliurament

Aquest mecanisme garanteix objectivitat i transparència.
