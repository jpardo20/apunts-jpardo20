# Activitat: Transformació de dades en HTML <kbd>0373-ae009</kbd>

## Objectiu de la pràctica

L’objectiu d’aquesta pràctica és **consolidar l’ús de XSLT** per transformar dades XML en una representació HTML, reforçant la idea clau del mòdul:

> **XML = dades**  
> **XSLT = transformació**  
> **HTML = resultat visual**

En cap moment s’han de modificar les dades originals.

---

## Material de partida

Treballarem amb els fitxers següents:

- `llibres.xml` → conté les dades
- `llibres.xsl` → defineix com es mostren les dades

Els dos fitxers han d’estar **a la mateixa carpeta**.


Contingut del fitxer `llibres.xml`
```xml
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="llibres.xsl"?>
<llibres>
  <llibre>
    <titol>Clean Code</titol>
    <autor>Robert C. Martin</autor>
    <preu>32</preu>
  </llibre>
  <llibre>
    <titol>Effective Java</titol>
    <autor>Joshua Bloch</autor>
    <preu>45</preu>
  </llibre>
</llibres>

```

Contingut del fitxer `llibres.xsl`

```xsl
<?xml version="1.0" encoding="UTF-8"?>

<xsl:stylesheet version="1.0"
    xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

    <xsl:template match="/">
        <html>
            <body>
                <h1>Llista de llibres</h1>
                <xsl:for-each select="llibres/llibre">
                    <ul>
                        <xsl:if test="preu &gt; 40">
                            <li>
                                <xsl:value-of select="titol"/>
                            </li>
                        </xsl:if>
                    </ul>
                </xsl:for-each>
            </body>
        </html>
    </xsl:template>

</xsl:stylesheet>
```

---

## Primera part: Versió `table`

Has de crear una transformació XSLT que mostri el contingut de `llibres.xml` en forma de **taula HTML** amb les columnes següents:

- **Títol**
- **Autor**
- **Preu**

Requisits:
- La taula ha d’estar dins d’un document HTML generat per XSLT.
- Cada llibre ha d’aparèixer en una fila diferent.
- **No es pot modificar l’arxiu `llibres.xml`.**
- La transformació s’ha de fer amb:
  - `xsl:template`
  - `xsl:for-each`
  - `xsl:value-of`

---

## Segona part: Filtrar llibres per preu

Modifica la transformació perquè **només es mostrin els llibres amb un preu superior a 40**.

Pista:

- Fes servir una condició amb `xsl:if`.

---

## Tercera part: Millora visual (HTML)

Afegeix una petita millora visual a la taula, per exemple:

- el preu en **negreta**
- o qualsevol altre detall HTML senzill

No cal afegir CSS. Només HTML bàsic.

---

## Lliurament

Has de lliurar **un únic fitxer comprimit (`.zip` o `.rar`)** amb:

- `llibres.xml`
- `llibres.xsl`

El codi ha de ser:

- clar
- ben indentat
- funcional
