## Colors extraterrestres #1:

Imagineu-vos que un extraterrestre acaba de ser abatut en un joc.

Creeu una variable anomenada **`alien_color`** i assigneu-li un valor de '**`verd`**', '**`groc`**' o 'vermell'.

• Escriviu una instrucció **`if`** per comprovar si el color de l'extraterrestre és **`verd`**.
Si ho és, escriviu un missatge que el jugador acaba de guanyar 5 punts.

• Escriviu una versió d'aquest programa que superi la prova **`if`** i una altra que falli.
    La versió que falla no tindrà cap sortida.

## Colors extraterrestres #2:

Trieu un color per a un extraterrestre com vau fer a l'exercici **`Colors extraterrestres #1`** i escriviu una cadena **`if-else`**.
• Si el color de l'extraterrestre és **`verd`**, escriviu una instrucció que el jugador acaba de guanyar
**5 punts per disparar a l'extraterrestre**.

• Si el color de l'extraterrestre no és **`verd`**, escriviu una instrucció que el jugador acaba de guanyar
**10 punts**.

• Escriviu una versió d'aquest programa que executi el bloc **`if`** i una altra que executi el bloc **`else`**.
