# apunts-jpardo

Portal de materials, apunts i recursos docents d'Informàtica per a Formació Professional.

Aquest repositori recull materials elaborats per a diferents mòduls dels cicles de la família d'Informàtica i Comunicacions, especialment SMX, DAM i DAW. Inclou apunts, activitats, presentacions, recursos de suport i materials pensats per treballar a l'aula amb metodologies actives.

## Objectiu

L'objectiu del projecte és disposar d'un espai centralitzat, reutilitzable i publicable amb materials docents propis, organitzats de manera clara i fàcilment accessibles per a l'alumnat.

## Contingut

El repositori inclou materials relacionats amb àmbits com:

- Muntatge i manteniment d'equips
- Sistemes informàtics
- Llenguatges de marques
- Bases de dades
- Programació
- Recursos transversals per a l'aula
- Activitats i materials de suport

## Tecnologies i eines

- HTML, CSS i JavaScript
- Markdown
- GitHub Pages
- Reveal.js per a presentacions
- Recursos interactius i formularis web

## Estat del projecte

Projecte en evolució contínua. El repositori s'utilitza com a espai de treball i publicació de materials docents, per tant alguns continguts poden estar en procés de revisió o ampliació.

## Llicència

El projecte inclou llicències separades per al codi i per als continguts docents. Consulta els fitxers de llicència del repositori per a més informació.
